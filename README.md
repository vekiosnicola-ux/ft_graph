# 🏛️ ft_graph: The 42 Piscine Knowledge Vault

A highly structured, evidence-based, and AI-optimized Obsidian vault designed to conquer the 42 Piscine.

## 🌟 Features
- **Clean Architecture**: Standardized folder structure (Modules, Labs, Exams, Concepts).
- **Cognitive Science Colors**: Evidence-based color coding (Orange=Flashcards, Blue=Modules, Amber=Concepts) for maximum retention.
- **AI-Ready**: Fully indexed for **Smart Connections**, allowing vector-search lookup across all C concepts.
- **Layered Graph View**: Multi-depth dependency visualization of C functions and architecture.
- **Comprehensive Docs**: Includes converted 42 CheatSheet manuals and 50+ function snippets.

## 🛠️ Requirements
- [Obsidian](https://obsidian.md/)
- Recommended Plugins (included in config):
  - Dataview
  - Smart Connections
  - Extended Graph
  - Iconic
  - Code Styler

## 📈 Graph Legend
- 🟠 **Flashcards**: Active recall nodes.
- 🔵 **Modules**: Core C assignments.
- 🟡 **Concepts**: Underlying C logic (Pointers, Memory, etc.).
- 🔴 **Exams**: High-pressure assessment notes.
- 🟢 **Manual**: Reference snippets and documentation.
